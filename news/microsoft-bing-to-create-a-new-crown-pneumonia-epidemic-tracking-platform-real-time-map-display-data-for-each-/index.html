開發者自製出 Google Maps 工具，讓你查是否接觸武漢肺炎病例

這個由開發者 yjlou 發表的「武漢肺炎歷史軌跡比對」工具，可以用 Google Maps 定位紀錄比對你和武漢肺炎（COVID-19）案例過去路徑，進而判斷自己的風險程度。

作者使用台灣和韓國政府已公開的案例歷史軌跡，不過使用者端必需開啟 Google Maps 的定位時間軸功能，才會有歷史路徑紀錄。開啟連結後，就能在網頁介面選擇案例的歷史軌跡檔案，並自動跟既有案例路徑紀錄比對。

由於這牽涉到定位紀錄，會有隱私外洩疑慮，因此作者也在頁面上強調一切比對都在本地端進行，不會上傳使用者的定位紀錄。2019-nCov 的網頁全部開源，在 Github 公開程式碼，也歡迎其他開發者加入專案。根據說明文件，目前僅有 2 位工程師參與開發。

目前實測看來，此工具沒有要求額外的權限，靠著已經下載的檔案資料，在本地端比對。

作者並附上案例資訊做成的 Google 「我的地圖」，且有詳細的說明頁面，也開放信箱供聯繫。

在說明頁面中，作者提到此計畫的緣起是看到有些政府有公開案例的軌跡，供民眾自我評估風險，然而隨著案例增多，去過的地點與時間資訊變得難以用人工比對，因此希望透過自動化的方式來處理。
